eXo GUI is a desktop graphical interface for eXo AI, a voice-activated personal assistant designed to centralize system control, utilities, and services into a single platform. It allows users to interact with the assistant using text commands and voice commands, with partial offline functionality available when AI features are disabled.

Through eXo GUI, users can manage reminders, control system actions, access weather updates, news, calendar details and click screenshots. It also supports IoT device control, application launching, theme switching, and AI-assisted responses powered by multiple APIs. Voice-based desktop control remains active while eXo operates in the background from the system tray(can be disabled).

eXo GUI is built to be functional, extensible, and focused, acting as the control hub for eXo AI where automation, information, and system interaction come together in one unified desktop interface.
